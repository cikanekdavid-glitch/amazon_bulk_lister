# Amazon Bulk Listing Generator

Generate Amazon bulk listing flat files using AI (Claude API). Paste your product names in Czech, get optimized English Amazon titles and bullet points.

## Use the App

**Live version:** https://amazon-bulk-lister.vercel.app

## Deploy Your Own Copy

Click the button below to get your own version with your own URL — no coding required:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/cikanekdavid-glitch/amazon_bulk_lister)

After deploying, you can edit it with Claude Code:
1. Vercel will fork the repo to your GitHub automatically
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/amazon_bulk_lister.git`
3. Open it with Claude Code and make changes
4. Push to `main` — your version auto-deploys
